# Game Template
This game template is meant to contain some general elements for a game in godot, such as menus and options. It also contains a game hierarchy, examplified through a small 2D platformer game.

# Licensed Assets
The game part of the template relies on some downloaded assets. To make this work properly, you will have to also download these assets and export them into the correct locations according to the following list:
* Free Green Zone Tileset Pixel Art:
  * Download the assets from https://craftpix.net/freebies/free-green-zone-tileset-pixel-art/?srsltid=AfmBOorc7-ZWiW3H1ShzoRUZWSZYpGJ7tKAqzU3bGsGTM_73JBfY7gHT
  * export the "Objects.png", "Tileset.png", and "Trees.png" files to the res://game/levels/assets/tilesets/ folder.
